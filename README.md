# 🛠️ Upload a File to AWS S3 Using Python (`boto3`)

This project demonstrates how to upload a file to an Amazon S3 bucket using Python and the `boto3` library.

---

## 📦 Requirements

- Python 3.x
- `boto3` (install via pip)
- An AWS account with:
  - Access Key ID
  - Secret Access Key
  - A valid S3 bucket

---
